# Hallmark gene sets (Human) from MSigDB

A two-column data frame mapping MSigDB Hallmark gene-set names to human
gene symbols, as retrieved from MSigDB v2024.1 via the msigdbr package.
The `"HALLMARK_"` prefix has been stripped from term names for brevity.

## Usage

``` r
hallmark_t2g
```

## Format

A data frame with 7,322 rows and 2 columns:

- term:

  Character. Hallmark gene-set name (`"HALLMARK_"` prefix removed).

- gene:

  Character. HGNC gene symbol.

## Source

Retrieved with `msigdbr(species = "Homo sapiens", collection = "H")` and
regenerated by `inst/prepare_data/hallmark_t2g.R`.

## Details

This object is provided as bundled package data so that
[`run_gsea`](https://statgencore.github.io/statplot/reference/run_gsea.md),
[`plot_pathways`](https://statgencore.github.io/statplot/reference/plot_pathways.md),
and
[`plot_pathway_correlation_network`](https://statgencore.github.io/statplot/reference/plot_pathway_correlation_network.md)
examples can run without an internet connection or an msigdbr
installation.
