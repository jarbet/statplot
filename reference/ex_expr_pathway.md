# Example expression matrix for pathway correlation network plots

A 20-gene × 40-sample simulated expression matrix used in
[`plot_pathway_correlation_network`](https://statgencore.github.io/statplot/reference/plot_pathway_correlation_network.md)
examples. Genes are drawn from the `MYC_TARGETS_V1` Hallmark pathway.
Three small correlated gene modules are embedded (within-module Pearson
r ≈ 0.9) alongside six independent genes, giving a realistic sparse
network.

## Usage

``` r
ex_expr_pathway
```

## Format

A numeric matrix with 20 rows (genes) and 40 columns (samples). Row
names are HGNC gene symbols; column names are `S1`–`S40`.

## Source

Generated by `inst/prepare_data/ex_pathway_data.R`.
