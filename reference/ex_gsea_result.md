# Example GSEA result for plot_pathways and plot_pathway_correlation_network

A pre-computed GSEA result generated by
[`run_gsea()`](https://github.com/jarbet/statplot/reference/run_gsea.md)
using a random gene-level statistic vector drawn from the
[`hallmark_t2g`](https://github.com/jarbet/statplot/reference/hallmark_t2g.md)
gene universe. Bundled so that
[`plot_pathways()`](https://github.com/jarbet/statplot/reference/plot_pathways.md)
examples can run without requiring clusterProfiler to be installed or an
internet connection.

## Usage

``` r
ex_gsea_result
```

## Format

A list with three elements as returned by
[`run_gsea()`](https://github.com/jarbet/statplot/reference/run_gsea.md):

- `gsea_result`:

  A `gseaResult` object.

- `gene_vec`:

  Named numeric vector of gene-level statistics (sorted decreasing).

- `term2gene`:

  Data frame with columns `term` and `gene`.
