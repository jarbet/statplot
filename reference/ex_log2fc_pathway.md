# Example log2 fold-change vector for pathway correlation network plots

A named numeric vector of simulated log2 fold changes for the 20 genes
in
[`ex_expr_pathway`](https://github.com/jarbet/statplot/reference/ex_expr_pathway.md),
used to color nodes in
[`plot_pathway_correlation_network`](https://github.com/jarbet/statplot/reference/plot_pathway_correlation_network.md)
examples.

## Usage

``` r
ex_log2fc_pathway
```

## Format

A named numeric vector of length 20. Names are HGNC gene symbols
matching the row names of
[`ex_expr_pathway`](https://github.com/jarbet/statplot/reference/ex_expr_pathway.md).

## Source

Generated by `inst/prepare_data/ex_pathway_data.R`.
